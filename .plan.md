# PEzor Package Plan

## Overview
Package PEzor v3.3.0 as a Nix derivation in `/mnt/nas/NixOS/pkgs/pezor/package.nix`.

PEzor is a bash script (`PEzor.sh`) that orchestrates several tools to pack shellcode and PE files. It references its own source directory (`$INSTALL_DIR`) for C/C++ source files, headers, and the `deps/inline_syscall` dependency.

## Dependencies

PEzor.sh requires these tools on `$PATH`:
1. **`x86_64-w64-mingw32-clang` / `clang++`** — wclang (clang wrapper for mingw cross-compilation)
2. **`donut`** — already packaged at `pkgs/donut/`
3. **`sgn`** — Go binary, pre-built release available from GitHub
4. **`mcs`** — Mono C# compiler (available in nixpkgs as `mono`)
5. **`xortool-xor`** — Python pip package `xortool`
6. **`cowsay`** — optional, in nixpkgs

### Sub-packages needed
- **wclang**: Build from source (cmake + make, wraps clang for mingw-w64 targets). Needs `clang`, `llvm`, and `mingw_w64` headers/libs.
- **sgn**: Fetch pre-built Linux binary from GitHub release (similar to sliver pattern)
- **xortool**: Build as a Python package from PyPI using `python3Packages.buildPythonPackage`

## Approach

### 1. Build wclang sub-derivation
- `fetchFromGitHub` from `tpoechtrager/wclang`
- Build with cmake, needs `clang`, `llvm`, `pkgsCross.mingwW64` for the cross sysroot
- Produces `x86_64-w64-mingw32-clang`, `x86_64-w64-mingw32-clang++`, etc.

### 2. Fetch sgn pre-built binary
- Fetch `sgn_linux_amd64_2.0.1.zip` from GitHub releases
- Unzip and install to `$out/bin/sgn`

### 3. Build xortool from PyPI
- Use `python3Packages.buildPythonPackage` with fetchPypi

### 4. Main PEzor package
- `fetchFromGitHub` from `phra/PEzor`
- Also fetch inline_syscall dep at the pinned commit and patch it in
- Patch `PEzor.sh`: replace `INSTALL_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"` with the Nix store path
- `makeWrapper` on `PEzor.sh` → `$out/bin/PEzor` with PATH containing all deps
- Install all source files (`.cpp`, `.hpp`, `deps/`, `dotnet/`, etc.) to `$out/share/pezor/`
- Set `INSTALL_DIR` in the wrapper to `$out/share/pezor`

## Files to create
- `/mnt/nas/NixOS/pkgs/pezor/package.nix` — main package with all sub-derivations inline
